# Task Management System

This is a simple web application to assist a user in managing tasks.
A user can:
    - Create an account 
    - Log in to their account
    - View all their tasks
    - Edit a task
    - Create a new task
    - Delete a task
    - Mark a task as completed

##Technologies Used:
###Backend:
* Ruby & Rails framework
* Sqlite3 Database

###Frontend
* React 
* CSS 
* Bootstrap
